Segregation IDL

Pour lancer l'application il suffit de lancer le fichier
"make_stuff.sh" avec :

sh ./make_stuff.sh

rajouter l'argument 0 pour l'application Fish et 1 pour l'application Segregation

Surtout fermer l'application avec le bouton "stop" pour avoir de bon
graphiques de sortie

Les graphiques seront alors générés avec gnuplot dans le dossier
de lancement
